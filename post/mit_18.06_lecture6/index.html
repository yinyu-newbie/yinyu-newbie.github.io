[MIT 18.06 线性代数 lecture6] - 列空间和零空间

2019-07-12 linear algebra math mit 18.06

上节课提到了向量空间的概念，顺便稍微提了下子空间以及列空间。本节课继续学习列空间，以及零空间。

回顾子空间

向量空间指的是在线性组合下是封闭的一组向量：向量空间中的任意两个向量 v\boldsymbol vv 和 w\boldsymbol ww ，在任取两个实数 ccc 和 ddd 的情况下，cv×dwc\boldsymbol v\times d\boldsymbol wcv×dw 同样在该向量空间中。

假设平面 PPP 和直线 LLL 均经过原点 [000]\begin{bmatrix} 0 \\ 0 \\ 0 \end{bmatrix}⎣⎡​000​⎦⎤​ ，则 PPP 和 LLL 均是 R3\mathbb{R}^3R3 的子空间。两者的并集 P∪LP\cup LP∪L 通常不是子空间，因为 PPP 中的向量和 LLL 中的向量的和可能并不在 P∪LP\cup LP∪L 中。而两个子空间 SSS 和 TTT 的并集 S∩TS\cap TS∩T 是子空间，根据 SSS 和 TTT 在线性组合下封闭的性质，容易证明两者交集是子空间。

AAA 的列空间

矩阵 AAA 的列空间（column space）是 AAA 的列向量的所有线性组合得到的向量空间。

求解 Ax=bA\boldsymbol x=\boldsymbol bAx=b

给定一个矩阵 AAA ，b\boldsymbol{b}b 取什么值时，存在解 x\boldsymbol{x}x 。

Ax=bA\boldsymbol{x}=\boldsymbol{b}Ax=b 不可能对每个 b\boldsymbol{b}b 都存在解，假设存在解 x\boldsymbol{x}x ，则 b\boldsymbol{b}b 必须是 AAA 的列向量的线性组合，仅仅 3 个列通过线性组合无法填充满整个四维向量空间，因此肯定存在某些向量 b\boldsymbol{b}b 无法通过 AAA 的列向量线性组合得到。

问题：什么样的 b′\boldsymbol{b'}b′ 使得 Ax=bA\boldsymbol{x}=\boldsymbol{b}Ax=b 有解？ 一个有用的方法是：直接选择不同的 x\boldsymbol{x}x 得到 b=Ax\boldsymbol{b}=A\boldsymbol{x}b=Ax 。当 b\boldsymbol{b}b 是 AAA 的列空间中的向量时，等式 Ax=bA\boldsymbol{x}=\boldsymbol{b}Ax=b 存在解。 上面的矩阵 AAA 中，第 3 列是前 2 列之和，对子空间的构建没有作用，AAA 的列空间是 R4\mathbb{R}^4R4 中的二维子空间。
